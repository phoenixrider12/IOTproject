Gas Sensor Predictions


About the project:

In this project, the development of a disaster preparedness and forecasting system is presented, which integrates real-time sensor data and time-sensitive computing to mitigate the effects of disasters. The system follows a service-oriented IoT architecture and focuses on early warning through machine learning (ML) algorithms implemented on a cloud server. It incorporates a triple modular redundancy fault tolerance scheme for the sensor network to ensure the availability of reliable and fault-free data for accurate predictions. The ML algorithm is applied to three sensor parameters (MQ4, MQ7, and force sensing resistor) on the AWS cloud, achieving low absolute error percentages between actual and predicted values, thus demonstrating the system's accuracy.

The project not only provides real-time monitoring and warning but also serves as a decision support system for disaster mitigation. It leverages IoT concepts to track plant behavior, with a strong emphasis on the development of a robust sensor network. Fault tolerance is a critical aspect, especially in safety-critical IoT applications. The proposed system offers a layer-level IoT architecture suitable for underground mines, where sensor data from various nodes are sent to a gateway node, and from there, to the cloud server for ML-based prediction and decision making. A fault-tolerant mechanism, employing triple modular redundancy, ensures uninterrupted, fault-free data transmission to the cloud, ultimately facilitating accurate predictions of plant conditions.

The primary contributions of this work include the design of a sensor network using Raspberry Pi gateway nodes, the development of a layer-level IoT architecture for EWS, and the use of AWS cloud for sensor data processing and prediction. Additionally, a fault-tolerant sensor network is designed to enhance reliability in data transmission and prediction. This project addresses the critical need for early warning and decision support systems in disaster preparedness, offering a reliable and fault-tolerant IoT-based solution for predictive monitoring in various applications, with a particular focus on underground mines.
